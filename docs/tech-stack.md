# Stack Técnico de Produção

## Geração de Imagem
- **Engine Principal:** ComfyUI
- **Modelos Preferidos:** Flux (realismo), QwenImage Edit (edição precisa).
- **Upscale:** ImageZ Turbo.
- **Dica para Prompts:** Priorizar descrições de iluminação (volumetric, softbox, natural light) e textura (veios, cristalização, polido/honed).

## Vídeo
- Wan2.2 e LTX2 para motion sutil (parallax, fumaça, luz).
